# fraudDetection
Fraud detection for BBD internal competition hosted at: https://www.kaggle.com/competitions/bbd-payments-hackathon/leaderboard
## We placed first in the competition.
Using an ensemble of XGBoost and a random forest regressor with 2 different feature sets.
The most powerful features created were an average spend over the last week and month as we as a risk value for each terminal over the same period of time.
## The final work used for the submission is in the presentation folder.
